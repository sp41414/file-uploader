# File Uploader

## Features:
1. Authentication
2. File uploading to Supabase
3. Views and styling with ejs and vanilla css
4. File and folder organization with CRUD
5. Recursively delete and create folders within folders and files within folders

## Steps:
1. Run `npm i`
2. Create a new postgres database
3. Go to .env and fill in your (secret) information
4. Run `npm run build` and `npm run db` and then run `npm run dev`
5. Go to http://localhost:3000 or whatever the port is you put in .env
